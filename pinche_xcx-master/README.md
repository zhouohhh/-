
## 项目说明

寻车，达成，寻同路人，拼车出行，绿色环保，本项目的目的是为了方便同程出行的用户，车主乘客自主发布拼车信息，并实现自能预约通知，使拼车更加方便快捷。

如果您使用的无appid测试,项目将无法正常登陆,所以如果您希望完全正常使用请添加appid

`因涉及到账号资质问题，此小程序并没有上线`
